# SplitApp

The Post method with endpoint /users consumes json:
EX.
[
{
"name": "Room rent",
"amount":"10000",
"paid_by": "ram",
"split_by": "equal"
},
{
"name": "EB bill",
"amount": "1000",
"paid_by": "kumar",
"split_by": "equal"
}]


The Get Method with endpoint /users returns List of strings as output.
